Vue3 + Vite + tailwind css + daisyUi 实现自定义分页器Pagination组件

tailwind css + daisyUi 真是太棒了, 效果如下：

快速开始

html 基本构成如下

<<（上一页按钮） 一直保持显示状态，当前页是第一页时禁用
1 （第一页按钮） 当中间连续按钮的起始键大于2时显示（当前页码数 - 连续页码量的一半 >= 2）
… / <<（省略号/前箭头） 当中间连续按钮的起始键大于3时显示
x x+1 … 中间连续数字
… / >>（省略号/后箭头） 当中间连续按钮的结束键 <= 总页数-2
T 最后一页按钮
>>（下一页按钮） 一直保持显示状态，当前页是最后一页时禁用
Go to 跳转输入框

props 基本构成如下

total 总条数
size 每页显示条数
continues 中间连续页码数量
